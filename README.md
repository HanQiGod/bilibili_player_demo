# bilibili_player_demo

基于 Flutter 的 B 站视频播放 demo，组合方案如下：

- `bilibili_api`：获取视频信息、分 P、CID 和播放地址
- `media_kit` + `media_kit_video`：播放视频
- `ns_danmaku`：渲染和同步弹幕

## 功能

- 支持输入 `BV`、`av` 或完整的 B 站视频链接
- 自动获取视频信息并显示封面、UP 主、播放量、点赞量
- 支持多分 P 视频切换
- 优先尝试 DASH 音视频流
- 如果当前视频拿不到可用 DASH 清单，会自动回退到 MP4 单流
- 通过官方 XML 弹幕接口拉取弹幕并按播放器时间轴同步显示
- 支持弹幕透明度、字号、滚动时长、顶部/底部/滚动弹幕开关

默认示例视频 ID 是 `BV1xx411c79H`，也可以自行输入任意公开视频测试。

## 说明

- 当前 demo 以“能直接播放公开视频”为目标，没有接入扫码登录。
- 未登录场景下，代码优先请求较低清晰度，以提高公开视频的可播放成功率。
- 对 DASH 流的处理方式是：从播放地址接口读取原始 dash 数据，动态生成 MPD 清单，再交给 `media_kit` 播放。
- 如果接口没有返回足够的 DASH 元数据，就自动回退到 MP4。
